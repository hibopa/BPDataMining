## 上传ppt

1.更改目录到当前仓库文件夹下(cd 目标文件夹/)

2.复制ppt到仓库文件中

3.使用git add ppt名称 添加文件到库

4.使用git commit -m "注释"提交注释

5.使用git push origin main 提交上传
